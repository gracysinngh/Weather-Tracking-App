# Weather-Tracking-App

A Python desktop application that fetches real-time weather data using the **OpenWeatherMap API** and displays it in a simple, interactive **Tkinter GUI**.  

## ✨ Features
- Dropdown menu with popular Indian cities (Delhi, Mumbai, Kolkata, Bangalore, etc.)
- Fetches live weather data (climate, description, temperature, pressure)
- Temperature converted from Kelvin to Celsius
- Clean and responsive GUI with labels and buttons
- Easy-to-use interface for beginners

## ⚙️ Tech Stack
- **Python**: Core programming language
- **Tkinter**: GUI framework for building the desktop interface
- **Requests**: For making HTTP requests to the weather API
- **OpenWeatherMap API**: Source of real-time weather data
